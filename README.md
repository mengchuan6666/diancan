# 在线点餐小程序

#### 1 介绍
🔥🔥🔥本项目完全开源，采用的技术是SpringBoot+Vue+Mysql+Mybatis+uniapp，项目非常的nice，带详细的部署教程,系统包括前后端、移动端，用户可以在移动端或者Web端进行下单。可以用于学习或者二次开发，感谢提出宝贵的建议，技术交流微：fly996868
